# Stock_Analyzer 

A comprehensive stock analysis platform built with React, featuring real-time data, technical indicators, portfolio management, and advanced charting.


## Features

- 📈 Real-time stock charts with multiple timeframes
- 🔍 Technical indicators (RSI, MACD, Bollinger Bands, Moving Averages)
- 💼 Portfolio management with profit/loss tracking
- 📰 Latest stock news and market updates
- 📊 Financial statements and company information
- 🎯 Stock comparison tool
- 📱 Fully responsive design
- 🌙 Dark/Light theme support
